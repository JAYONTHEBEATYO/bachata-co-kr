# AGENTS.md

## Korean Copy Rules

- Write for dancers first, not for editors or crawlers.
- Prefer concrete Korean dance-scene terms: `강습`, `오픈강습`, `정모`, `소셜`, `왕초보`, `입문`, `초급`, `워크숍`, `내한`, `양도`.
- Avoid public-facing internal terms: `허브`, `레이더`, `큐`, `후보`, `신호`, `승격`, `검수`, `워치리스트`, `Graph API`, `oEmbed`, `자동화`.
- Avoid self-referential SEO copy such as `검색엔진이 좋아하는`, `검색 의도`, `플랫폼으로 성장`, `깊은 한국어 콘텐츠`.
- Titles and descriptions should lead with user value: date, location, class, social, event, team, venue, or safety point.
- Do not copy Instagram captions or private community posts. Use public links and rewrite in original Korean.
- Keep section labels reader-facing: `볼 포인트`, `영상 안내`, `관련 검색어`, `같이 보기`, `공식 링크`.

Run `node tools/audit-korean-copy.mjs` before publishing copy-heavy changes.
