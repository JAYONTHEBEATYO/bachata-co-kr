Policy

수집과 발행의 선

자동 수집은 후보를 만드는 데까지만 씁니다. 실제 글은 출처를 확인하고 한국 바차타 독자가 읽기 좋은 맥락으로 다시 작성합니다.

Instagram Graph API

프로페셔널 계정 기반 해시태그, 멘션, owned media 신호 확인

Business 또는 Creator 계정 권한과 앱 리뷰가 필요하고, 해시태그 조회는 7일 30개 제한을 고려해야 합니다.

원본 캡션·사진을 복제하지 않고 링크, 계정명, 날짜, 관련 페이지로만 발행 후보를 만듭니다.

해시태그는 검색 의도와 취재 방향으로만 쓰고, 게시물 본문은 편집자가 직접 확인합니다.

기사 안에 꼭 필요한 공개 게시물만 임베드하고, 설명 문장은 자체 작성합니다.
