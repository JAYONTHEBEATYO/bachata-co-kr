Policy

복제하지 않는 운영 원칙

자동화는 수집과 후보 정렬까지만 담당합니다. 발행문은 출처를 보고 한국 독자에게 맞는 문장으로 다시 편집합니다.

Rule 1

복제 금지

인스타 캡션과 이미지는 무단 복제하지 않고 링크, 핸들, 태그, 공개 임베드 가능 여부만 기록합니다.

Rule 2

API 분리

Instagram Graph API는 professional account, hashtag search, oEmbed 용도와 권한이 다르므로 수집 목적별로 분리합니다.

Rule 3

교차 확인

행사·내한·워크숍은 공식 페이지, 주최자 인스타, 유튜브 아카이브 중 두 개 이상으로 교차 확인한 뒤 기사화합니다.

Rule 4

검수 발행

매일 자동 브리프는 후보 수집이고, evergreen 기사와 커뮤니티 공지는 편집 검수 후 발행합니다.
